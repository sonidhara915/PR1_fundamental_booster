# 🌟Fundamental Booster Project

# 🌟 Interactive Personal Data Collector

> 🚀 **A beginner-friendly Python project that collects user information, explores data types, and calculates the approximate birth year**

---

# 📌 Project Overview

The **Interactive Personal Data Collector** is a simple Python application that demonstrates the fundamentals of Python programming through an interactive console program.

It collects basic information from the user and displays additional details such as the **data type**, **memory address**, and **approximate birth year**.

---

# ✨ Features

✅ Interactive user input

✅ Accepts multiple data types

✅ Displays stored values

✅ Shows variable data types

✅ Displays memory addresses using `id()`

✅ Calculates approximate birth year

✅ Beginner-friendly project

---

# 📥 User Inputs

The program asks the user to enter:

👤 **Name**

🎂 **Age**

📏 **Height (in meters)**

⭐ **Favorite Number**

---

# 📤 Program Output

After collecting the information, the program displays:

🔹 Name

🔹 Age

🔹 Height

🔹 Favorite Number

🔹 Data type of each value

🔹 Memory address of each variable

🔹 Approximate birth year

🔹 Thank-you message

# 🛠 Functions Used

| 🔧 Function | 📖 Purpose |
|-------------|------------|
| `print()` | Displays output |
| `input()` | Accepts user input |
| `int()` | Converts input into Integer |
| `float()` | Converts input into Float |
| `type()` | Returns the data type |
| `id()` | Returns the memory address |
